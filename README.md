# 🚀 DevHaat — Digital Tools Marketplace

## 📌 Project Overview

**DevHaat** is a modern digital tools buying platform where developers can explore and purchase useful resources like UI kits, templates, and productivity tools.
The goal of this project is to provide a clean, user-friendly interface with smooth cart functionality and responsive design.

## ⚙️ Technologies Used

- ⚛️ React.js
- 🎨 Tailwind CSS & DaisyUI
- 📜 JavaScript (ES6+)
- 🔔 React Hot Toast
- 📦 JSON (for product data)

---

## ✨ Key Features

### 🛒 Smart Cart System

- Add products to cart
- Remove individual items
- Clear cart using "Proceed to Checkout"
- Real-time cart count update in navbar

---

### 🔄 Product & Cart Toggle

- Switch between product list and cart view
- Default view shows all products
- Cart shows empty message when no items added

---

### 📱 Fully Responsive Design

- Optimized for mobile, tablet, and desktop
- Clean and modern UI layout

---

## 🔔 Notification System

All user actions like:

- Add to cart
- Remove item
- Checkout

👉 are handled using **React Hot Toast** alerts.
